# Attendance Dashboard

This Streamlit app can be deployed to Render, Streamlit Community Cloud, or similar Python hosting services.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Render

1. Push this repository to GitHub.
2. Create a new Web Service on Render and connect the repository.
3. Use the existing Render configuration in render.yaml.
4. Render will install dependencies from requirements.txt and start the app with the provided command.
